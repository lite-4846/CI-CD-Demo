# ❓ Frequently Asked Questions

**Q1:** What is IRIS-Instant primarily used for?  
**A:** It’s designed to automate workflows and deliver modular services efficiently.

**Q2:** Can I run it without Docker or Jenkins?  
**A:** Yes, you can run the Angular and Spring Boot services locally. CI/CD is optional but recommended.

**Q3:** Is IRIS-Instant open source?  
**A:** Internal use only for now; future releases may include open access.

**Q4:** What database does it support?  
**A:** PostgreSQL is the default and fully supported.
